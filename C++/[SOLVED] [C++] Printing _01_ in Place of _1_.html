    Re: [C++] Printing "01" in Place of "1"

    This should work:
    Code:
    #include <iostream>
#include <iomanip>

int main()
{
   int i = 1;

   std::cout << std::setw(2) << std::setfill('0') << i << std::endl;
}
    setw == set width
    setfill == set unused spaces to the given character

    Read more here: http://www.cplusplus.com/reference/iostream/

    Re: [C++] Printing "01" in Place of "1"

    I would do it like this :

    Code:
    fprint(stdout, "%c%d", (i>0 &&i<=9)?'0':'', i);

or in pure c++

cout << (i>0 && i<=9)?'0':'' << i;
    The conditional operator outputs a '0' character if i is between 0 and 9.
